EITIx

EITIx is a SOC Control Center (CC), serving as an all-in-one solution. Security analysts monitor, analyze, and respond to security events. EITIx is a proprietary solution, developed and maintained by EaglesIdea Ltd. At present, EITIx Version 1.9.0 is integrated with Wazuh, the leader in open-source SIEM.

Benefits of EITIx

Below represents the main benefits users get after using EITIx SOC Control Center:

Everything in One Dashboard

Security team maintains multiple applications for multiple purposes. So, it becomes hard to login multiple apps daily to find security risks. But EITIx will provide all those features in a single application. This enables maintaining all the security operations in a single dashboard.

Threat Intel Automation

EITIx integrates the threat intelligence feeds such as MISP, NIST, VirusTotal, etc., to automate the malicious file, IP address, and website domain detection. So, the security team doesn’t need to scan IoCs in these threat intel feeds manually.

Artificial Intelligence (AI)

When you have millions of logs, it’s hard to tell if there are any phishing, malware, etc., logs. So, EITIx artificial intelligence (AI) enables users to view how many logs are related to which attack vectors. This also helps to identify threats that were not detected by your security detection rules.

Features of EITIx

EITIx SOC Control Center features are described below:

Security Log Analysis

EITIx is integrated with Wazuh SIEM. And EITIx users can analyze all the logs in EITIx. The main advantage is-EITIx provides you easy searching capability without using any complex query language i.e.,SQL etc.

Attack Surface: Playbook

Your organization may get million of logsdaily. Which becomes very hard to analyze and discover real threats.To solve this problem, EITIx provides a feature called ‘Playbook’ where you can create custom rules to get only specific alerts.

Artificial Intelligence(AI)

EITIx artificial intelligence(AI) enables users to view how many logs are related to which attack vectors.This helps to identify threats that were not detected by your own created security detection or, threat hunting rules.

Threat Intelligence Integration

EITIx integrates the threat intelligence feeds i.e., MISP, NIST, VirusTotal etc. to automate the malicious file, IP-address and website domain etection. So, security team oesn’t need to scan IoCs in these threat intel feeds manually.

Network Assets Monitoring

EITIx not only enables you to monitor the SIEM(Wazuh) agents status. Butalso, EITIx enables you to add any cloudor, network assets so that you can monitor all your assets i.e., API, database etc. in a single dashboard of EITIx.

Custom Report Generate: Quick-Report

EITIx users don’t need to manually create any audit report with a summarized data records. The ‘QuickReport’ feature let’s theusers generatea security report by scanning all the agents.

Custom Report Generate: Detail-Report

EITIx users,when need to create a big audit report, don’t need to manually create any audit report with all the detailed data records.The‘Detail-Report’ feature let’s the users generate a security report by scanning selected agents.

Vulnerability Management

EITIx modules always keep scanning your agents and logs to findout any possible CVE vulnerabilities.And when detected any vulnerability, EITIx provides you all possible details and risk severity of the vulnerability.

Incident Management: Ticket

EITIx is bundled with its own ticketing system. Users can create tickets for each security, assign appropriate users, divide an incident to multiple phases, comment out with evidence and attachments any manymore.

Indicators of Compromise

EITIx scans all the files, IP addresses and website domains that are accessed by users in your organization. EITIx does this by integrating with cyber threat intelligence platforms. And if anything malicious found, EITIx generates alert.

World Map

EITIx users can monitor which users or, employees are accessing which country’s servers, websites etc. The ‘WorldMap’ in EITIx dashboard enables this feature so that you can find connections to any suspicious geo locations.

Email and SMS Alert

EITIx enables users to get alerts via email and SMS both. From scheduled playbook reports to security incident management(ticket) related status, all are reported to appropriate users via his email and MS alerting system.
